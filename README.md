Setting up front-end

- `git clone https://github.com/csihda/adamant.git`
- `npm install`
- `npm start` or `yarn start`

Installing back-end
- `cd backend`
- `python -m venv venv`
- `./venv/Scripts/activate`
- `pip install -r requirements.txt`
- `cd ..`
- `yarn start-api`
